# 🚀 theGLabs — Portfólio

Site portfólio pessoal desenvolvido com **HTML**, **CSS** e **JavaScript** puro.

## ✨ Funcionalidades

- Robô 3D em CSS que segue o cursor do mouse
- Efeito glitch + highlight animado no texto
- Partículas interativas que reagem ao mouse
- Animações de entrada com fade-up em todas as seções
- Transição suave entre páginas
- Barra de progresso de scroll
- Cursor customizado
- Design responsivo com menu mobile
- Dark mode

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## 📄 Seções

- **Início** — apresentação, contato e redes sociais
- **Habilidades** — stack, barras de nível animadas e sobre mim
- **Projetos** — cards com link para deploy e GitHub
